# Europe Data Jobs Project

## Overview
Analyze European data job opportunities to uncover:
- Hotspots by geography
- Most demanded roles (Data Scientist, Analyst, etc.)
- Top skills
- Industry trends

## Folder Structure
- notebooks/: Colab notebooks for EDA and analysis
- data/: raw and processed datasets
- visuals/: plots and charts
- reports/: analysis summaries
- requirements.txt: packages used in Colab

## Project Status
Work in progress. All work tracked via GitHub + Colab.
